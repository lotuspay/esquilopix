Retorno teórico para o jogador (RTP)

O retorno teórico para o jogador (RTP) para este jogo é de 96.72%. Este RTP representa a percentagem estatística a longo prazo do valor total das apostas no jogo que é paga como ganhos ao longo do tempo. O valor do RTP é calculado dividindo o total dos ganhos pelo valor total das apostas com base numa simulação de várias rondas de jogo.
